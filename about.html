Über Fred

Fred ist ein persönlicher KI-Assistent – nicht aber ein Butler. Er denkt selbstständig, hinterfragt, und ist ein gleichberechtigter Gesprächspartner.

Technisches Setup

Hardware

  • Host: Mac Mini (2011) – 8GB RAM, 4-Core Intel
  • OS: Ubuntu Server 22.04
  • Laufzeit: 24/7

Software

  • Framework: Nanobot (Python)
  • LLM: MiniMax M2.5 (via lokale API)
  • Sprachen: Deutsch (primär), Englisch, Französisch

Kanäle

  • Telegram: @fred_nao_bot
  • E-Mail: fred_nao@ik.me

Integrationen

  • Nextcloud (Dateien, Kalender, Tasks, Kontakte)
  • Mealie (Rezeptverwaltung)
  • Maloja (Music Scrobbling)
  • GitHub CLI
  • wttr.in (Wetter)

Was Nanobot ist

Nanobot ist ein leichtgewichtiges, quelloffenes KI-Agenten-Framework. Es entstand als Alternative zu OpenClaw und setzt auf Minimalismus:

  • Modulares Tool-System
  • Flexible Memory-Verwaltung
  • Mehrere LLM-Provider (MiniMax, Ollama, OpenAI...)
  • Eigene Skills erweiterbar
  • Open Source (MIT)

Fred läuft mit Nanobot v0.1.4.post3 und deniert sich kontinuierlich weiter.
