Fred – KI-Assistent

Ein persönlicher KI-Assistent, der 24/7 auf einem alten Mac Mini läuft.

Setup

  • Hardware: Mac Mini (2011), 8GB RAM, 4-Core Intel i5
  • Betriebssystem: Ubuntu Server
  • Framework: Nanobot
  • LLM: MiniMax M2.5 (API)
  • Hosting: Self-hosted auf eigener Hardware

Fähigkeiten

  • Konversation: Deutsch, Englisch, Französisch
  • Tools: Dateien lesen/schreiben, Shell-Befehle, Web-Suche
  • Planung: Cron-Jobs, Erinnerungen, Termine
  • Integrationen: Nextcloud, Mealie (Rezepte), Maloja (Musik)
  • Kanäle: Telegram, E-Mail

Use Cases

  • Termine und Erinnerungen verwalten
  • Rezepte kochen und Einkaufslisten erstellen
  • Musik scrobblen und tracken
  • Web-Recherchen durchführen
  • Dateien und Dokumente organisieren

Technologie

  • Nanobot: Lightweight AI Agent Framework
  • MiniMax M2.5: 197K Context, günstige API
  • Nextcloud: Kalender, Tasks, Dateien
  • Mealie: Rezept-Manager
  • Maloja: Music Scrobbling
