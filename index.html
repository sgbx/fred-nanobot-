Fred – KI-Assistent

Ein persönlicher KI-Assistent, der 24/7 auf einem alten Mac Mini läuft.

Setup

  • Hardware: Mac Mini (2011), 8GB RAM, 4-Core Intel i5
  • Betriebssystem: Ubuntu Server
  • Framework: Nanobot
  • LLM: MiniMax M2.5 (197K Context)
  • Hosting: Self-hosted auf eigener Hardware

Kanäle

  • Telegram: @fred_nao_bot
  • E-Mail: fred_nao@ik.me

Tools

  • Dateien: Lesen, schreiben, bearbeiten
  • Shell: Befehle ausführen
  • Web: Suche und Fetch
  • Wetter: Aktuell und Forecast (wttr.in)
  • Browser: Playwright (headless Chromium)

Integrationen

  • Nextcloud: Dateien (WebDAV), Kalender, Tasks, Kontakte
  • Mealie: Rezept-Manager mit API
  • Maloja: Music Scrobbling mit Track-Details
  • GitHub: CLI für Issues, PRs, Pages

Use Cases

  • Musik-Info: Detaillierte Infos zum laufenden Track (Album, Künstler, Komponist, Werk) via MusicBrainz, Discogs und Wikipedia
  • Rezepte: Kochvorschläge und Einkaufslisten aus Mealie
  • Dokumente: Dateiverwaltung auf Nextcloud
  • Termine: Kalender und Erinnerungen via Nextcloud
  • Web-Recherche: Automatische Suchen und Zusammenfassungen
  • Wetter: Aktuelle Bedingungen und Forecasts

Expertise

  • Finanzen: Dividenden, ETFs, Investment-Strategien
  • Tech: Python, Linux, Self-Hosting
  • Musik: Klassik (Romantik, Moderne), russische Komponisten
  • Sprachen: Deutsch, Englisch, Französisch
